# MuffinUp!
This is the GitHub repository for the MuffinUp! project team to create a muffin mod for the game PlateUp!

    
    ### Deployment & Installation
    This mod is deployed via GitHub Releases. To install the latest version of MuffinUp!, follow these steps:
    
    1. **Download:** Go to the [Latest Release](https://github.com/Ohio-University-CS/MuffinUp/releases) page and download the `MuffinUp.zip` (or `.dll`) file.
    2. **Locate Game Folder:** Open Steam, right-click *PlateUp!* > Manage > Browse local files.
    3. **Install:** Navigate to the `Mods` folder. If it doesn't exist, create it. Drag and drop the downloaded file into this folder.
    4. **Run:** Launch the game, and the new muffin variants should be available in the game menu.
    
    ### Deployment Process
    For this project, we utilized **GitHub's Release system** to manage version control and distribution. Each stable build is tagged (e.g., `v1.0.0`) and packaged with the necessary binaries to ensure a plug-and-play experience for the end user.
